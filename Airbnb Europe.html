Explorez et utilisez l'application Airbnb Europe avec facilité !

1. Base de Données

La base de données fournit un aperçu complet des prix Airbnb dans certaines des villes européennes les plus populaires au cours de l’année 2021.
Nous avons décidé pour le développement de notre application de nous focaliser uniquement sur les villes : Paris, Amsterdam, Barcelone, Berlin, Rome.
L’application pouvant bien entendu être déclinée à toutes les autres.
Chaque annonce Airbnb est évaluée en fonction de divers attributs : le type de logement, les notes de propreté et de satisfaction, la capacité d’accueil, la distance du centre-ville, les coordonnées géographiques (latitude et longitude), la distance au métro ainsi que le prix du bien à louer.
Nous avons décidé de ne garder que les variables

La base de données brute est visible dans l'onglet 'Base de données' .

2. Prix Moyen par Ville

Consultez l'onglet 'Prix moyen' pour découvrir le graphique des prix moyens par ville, ainsi que le prix moyen en fonction des différentes caractéristiques du logement.

3. Carte Interactive

L'onglet 'Recherche' vous permet d'explorer les offres Airbnb par ville. Utilisez les filtres sur le côté pour affiner vos critères de recherche. La carte affiche des marqueurs colorés en fonction des prix. Survolez les marqueurs pour voir les prix exacts et les détails.

4. Prédiction du Prix

L'onglet 'Dépôt annonce' vous permet d'estimer le prix de location d'un bien Airbnb en fonction de certaines caractéristiques. Voici comment utiliser cette fonctionnalité :
• Choisir une ville : Sélectionnez la ville où se trouve le bien que vous souhaitez louer.
• Sélectionner le jour : Choisissez entre 'Weekend' et 'Weekdays' en fonction de vos besoins.
• Type de Logement : Sélectionnez le type de logement que vous souhaitez proposer, par exemple, Private room
• Capacité : Indiquez le nombre de personnes que votre bien peut accueillir.
• Localisation : Cliquez sur la carte pour sélectionner l'emplacement exact de votre bien.
Ensuite, l'application utilisera un modèle de prédiction de type Random Forest préalablement entraîné pour prédire le prix de location en fonction de ses caractéristiques.
